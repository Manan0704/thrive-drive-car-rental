# ThriveDrive - Car Rental Website

ThriveDrive is a web-based car rental system developed as a BCA final year project.

## 🚗 Features
- User Registration & Login
- Car Booking System
- Admin Dashboard
- Vehicle Management
- Booking History

## 🛠️ Technologies Used
- Laravel (PHP Framework)
- MongoDB
- HTML, CSS, JavaScript
